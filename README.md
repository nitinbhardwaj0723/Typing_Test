# ⌨️ Advanced Typing Test App

An advanced typing speed test web application built using HTML, CSS, and JavaScript. This app helps users improve their typing speed and accuracy with real-time feedback and a modern UI.

## 🚀 Features

- ⏱️ 60-second timer
- ⚡ Real-time WPM (Words Per Minute) calculation
- 🎯 Live accuracy tracking
- ✅ Correct & ❌ incorrect character highlighting
- 🔤 Current character focus indicator
- 🔄 Random text generation
- 🎨 Clean and modern dark UI
- 🔁 Restart and new text functionality

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
